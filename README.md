# Multi_Factor_V2.ipynb
一个基本的多因子分析的讲解框架

优点：
1、能够提取相关财务因子

2、能够灵活的设置因子数量

3、能够设置倒数

# Multi_Factor.py

多因子的策略实现，需要注意部分函数只能在
- 选股因子：6个已知方向的因子

- 数据处理：处理缺失值


选股权重：

- 因子升序从小到大分10组，第几组为所在组得分

- 因子降序从大到小分10组，第几组为所在组得分


  
- 选股的指数、模块：沪深300
  

- 调仓：每月（周）进行一次调仓选出20个排名靠前的股票

- 交易规则：卖出已持有的股票，买入新的股票池当中的股票
